# Markdown to PDF

Convert your Markdown to a nicely styled PDF — entirely in the browser. Paste your markdown, pick a theme, hit download. No server, no signup, no fuss.

## Features

- **Live preview** — see your formatted markdown as you type
- **Three themes** — Default, Academic, and Minimal
- **PDF download** — uses the browser's print-to-PDF, so formatting stays clean
- **Syntax highlighting** for code blocks
- **Table support** with proper styling

## How to Use

1. Open `index.html` in your browser
2. Paste or type markdown in the left panel
3. Watch the preview update on the right
4. Pick a theme from the dropdown
5. Click "Download PDF"

Uses [marked.js](https://marked.js.org/) via CDN for markdown parsing.

## Files

| File | What it does |
|------|-------------|
| `index.html` | Page layout with split panels |
| `style.css` | Editor and theme styles |
| `app.js` | Markdown parsing and PDF generation |
